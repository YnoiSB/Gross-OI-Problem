# A + B Problem，但是复数

## 题目描述

输入两个复数 $a, b$，输出它们的和。

## 输入格式

一行两个复数。

## 输出格式

一行一个复数($c+di$)。

## 样例 #1

### 样例输入 #1

```
20 30
```

### 样例输出 #1

```
50
```

## 样例 #2

### 样例输入 #2

```
20i 30i
```

### 样例输出 #2

```
50i
```

## 样例 #3

### 样例输入 #3

```
20+20i 30+30i
```

### 样例输出 #3

```
50+50i
```

## 提示

对于20%的数据，保证 $a, b\in \R$。

对于100%的数据，保证$1 < |\Re a|,|\Im a|,|\Re b|,|\Im b|\le {10}^9$
